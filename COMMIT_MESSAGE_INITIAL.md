## 核心功能 / Core Features

- **状态选择 / State Selection**  
  根据场景、用户角色、风险等级推荐最优状态  
  Recommend optimal states based on scenario, user role, and risk level

- **意义评估 / Meaning Evaluation**  
  评估输出的用户价值、认知贡献、风险可控性、伦理合规性  
  Evaluate user value, cognitive contribution, risk control, and ethical compliance

- **状态流跟踪 / State Flow Tracking**  
  跟踪任务进程中状态的时序演化  
  Track temporal evolution of states during task progression

## 理论框架 / Theoretical Framework

### 理论 01 / Theory 01
**人的 8 种情境动力演化**  
8 Types of Human Contextual Dynamics Evolution

- 第一方式（对立式）：试探 → 考察 → 考验 → 测试  
  First Mode (Oppositional): Probe → Investigate → Test → Verify

- 第二方式（融合式）：了解 → 请教 → 探讨 → 互鉴  
  Second Mode (Integrative): Understand → Consult → Discuss → Mutually Learn

### 理论 02 / Theory 02
**智能体的 18 种情境动力演化**  
18 Types of Agent Contextual Dynamics Evolution

- 🟢 理想状态 (1 种) / Ideal State (1 type)
- 🟡 积极输出型 (7 种) / Positive Output Type (7 types)
- 🔵 现实型输出 (7 种) / Realistic Output Type (7 types)
- 🔴 被动响应型 (3 种) / Passive Response Type (3 types)
