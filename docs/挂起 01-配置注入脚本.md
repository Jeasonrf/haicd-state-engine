## 🎯 任务概述

### 目标
创建配置注入脚本，使用户在授权后可以将 HAICD 技能集成到智能体的输出控制层，实现常驻状态选择和意义评估功能。

### 核心功能
1. **用户授权机制**：明确获取用户同意
2. **配置生成**：生成 HAICD 集成配置
3. **配置注入**：修改 OpenClaw 配置文件
4. **控制脚本**：提供启用/禁用/状态检查功能
5. **回滚机制**：随时可禁用，配置可逆

### 1. 用户授权机制
```python
def get_user_consent():
    """获取用户授权"""
    print("""
    HAICD 输出控制层集成授权
    
    本操作将：
    1. 将 HAICD 技能设置为常驻输出控制层
    2. HAICD 将参与所有智能体输出的处理
    3. 可以随时禁用此功能
    
    您是否授权？ (yes/no)
    """)
    consent = input().strip().lower()
    return consent == 'yes'
```
