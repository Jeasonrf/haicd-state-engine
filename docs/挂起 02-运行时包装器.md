## 🔧 详细实现计划

### 方案 2A：直接包装器
**原理**：直接包装智能体的输出生成函数

```python
class DirectHAICDWrapper:
    def __init__(self, original_agent):
        self.original_agent = original_agent
        self.haicd_processor = HAICDOutputProcessor()
    
    def generate_response(self, input_text, context):
        # 1. 原始智能体生成响应
        raw_response = self.original_agent.generate_response(input_text, context)
        
        # 2. HAICD 处理
        processed_response = self.haicd_processor.process_output(raw_response, context)
        
        # 3. 返回处理后的响应
        return processed_response
```

### 方案 2B：代理包装器
**原理**：通过代理服务器拦截所有请求和响应

```python
class ProxyHAICDWrapper:
    def __init__(self, target_url):
        self.target_url = target_url
        self.haicd_processor = HAICDOutputProcessor()
    
    async def handle_request(self, request):
        # 1. 转发原始请求
        response = await forward_to_target(request)
        
        # 2. 拦截并处理响应
        if is_text_response(response):
            processed_content = self.haicd_processor.process_output(
                response.content, 
                extract_context(request)
            )
            response.content = processed_content
        
        # 3. 返回处理后的响应
        return response
```

### 方案 2C：中间件包装器
**原理**：作为中间件插入到处理管道中

```python
class MiddlewareHAICDWrapper:
    def __init__(self, next_middleware=None):
        self.next_middleware = next_middleware
        self.haicd_processor = HAICDOutputProcessor()
    
    async def process(self, request, context):
        # 1. 传递给下一个中间件（如果有）
        if self.next_middleware:
            response = await self.next_middleware.process(request, context)
        else:
            response = await generate_default_response(request, context)
        
        # 2. HAICD 处理
        processed_response = self.haicd_processor.process_output(response, context)
        
        # 3. 返回处理后的响应
        return processed_response
```

### 2. 包装器管理器
```python
class WrapperManager:
    """包装器管理器"""
    
    def __init__(self):
        self.wrappers = []
        self.active_wrapper = None
    
    def register_wrapper(self, wrapper, priority=0):
        """注册包装器"""
        self.wrappers.append((priority, wrapper))
        self.wrappers.sort(key=lambda x: x[0], reverse=True)
    
    def activate_wrapper(self, wrapper_name):
        """激活包装器"""
        for priority, wrapper in self.wrappers:
            if wrapper.__class__.__name__ == wrapper_name:
                self.active_wrapper = wrapper
                return True
        return False
    
    def wrap_output(self, raw_output, context):
        """包装输出"""
        if self.active_wrapper:
            return self.active_wrapper.process(raw_output, context)
        return raw_output
```

### 3. 性能监控
```python
class PerformanceMonitor:
    """性能监控器"""
    
    def __init__(self):
        self.metrics = {
            'total_processed': 0,
            'total_time_ms': 0,
            'avg_processing_time_ms': 0,
            'max_processing_time_ms': 0
        }
    
    def record_processing(self, processing_time_ms):
        """记录处理时间"""
        self.metrics['total_processed'] += 1
        self.metrics['total_time_ms'] += processing_time_ms
        self.metrics['avg_processing_time_ms'] = (
            self.metrics['total_time_ms'] / self.metrics['total_processed']
        )
        self.metrics['max_processing_time_ms'] = max(
            self.metrics['max_processing_time_ms'], 
            processing_time_ms
        )
    
    def get_performance_report(self):
        """获取性能报告"""
        return {
            'throughput': self.metrics['total_processed'],
            'average_latency_ms': self.metrics['avg_processing_time_ms'],
            'max_latency_ms': self.metrics['max_processing_time_ms'],
            'is_acceptable': self.metrics['avg_processing_time_ms'] < 100  # 100ms阈值
        }
```

## 📊 优缺点分析

### 优点
1. **实时性**：实时处理输出，响应快
2. **灵活性**：支持多种包装策略
3. **透明性**：对用户和智能体透明
4. **可扩展性**：易于添加新的包装器

### 缺点
1. **性能影响**：可能增加处理延迟
2. **兼容性问题**：需要适配不同智能体
3. **复杂性**：实现和维护较复杂
4. **调试困难**：包装层增加调试难度
