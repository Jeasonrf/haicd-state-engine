## 情感动力学模型（TR-Dynamics）

### 动力学方程

$$\frac{dT}{dt} = \alpha(T) \cdot Q(t)$$
$$\frac{dR}{dt} = \beta(R) \cdot Q(t)$$

### 参数定义

| 参数 | 含义 | 取值范围 |
|------|------|---------|
| T(t) | 信任度 | [0, 1] |
| R(t) | 尊重度 | [0, 1] |
| Q(t) | 互动质量 | [-1, 1] |
| α(T) | 信任调整系数 | 非线性函数 |
| β(R) | 尊重调整系数 | 非线性函数 |

### 调整系数形式

**第一方式**（不信任前置）：
- α(T) = k₁·(1-T)^p
- β(R) = k₂·(1-R)^q

**第二方式**（信任前置）：
- α(T) = k₁·T^m
- β(R) = k₂·R^n

## 强化学习框架（ERL-Interact）

### 组件定义

| 组件 | 描述 |
|------|------|
| 状态空间 S | T(t), R(t), Q(t), 历史上下文，用户情绪 |
| 动作空间 A | 5 种回应策略（泛泛/引导/深度/共鸣/错误） |
| 奖励函数 R | Q(t) + 用户反馈评分 |
| 算法 | PPO（Proximal Policy Optimization） |
| 优化目标 | 最大化长期信任积累 |
