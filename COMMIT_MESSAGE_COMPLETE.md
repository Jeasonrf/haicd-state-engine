## 🚀 使用方式

### 快速开始
```bash
# 启用 HAICD 输出控制层
cd ~/.openclaw/skills/haicd-state-engine/scripts
./control/enable_haicd.sh

# 检查状态
./control/status_haicd.sh

# 测试输出处理
python3 haicd_output_processor.py

# 禁用集成
./control/disable_haicd.sh
```

### 配置清理（如果需要）
```bash
# 清理配置中的 integrations 键
python3 cleanup_integrations.py clean

# 检查配置有效性
python3 cleanup_integrations.py check
```
